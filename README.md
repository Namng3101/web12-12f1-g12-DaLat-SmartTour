# Đà Lạt SmartTour

## Chủ đề
Website giới thiệu du lịch Đà Lạt, bao gồm các mục: Hero video, About, Gallery, Tour List, Contact, Founders và Booking Modal.  
Dựng theo khung W3.CSS theo yêu cầu của bài thực hành.
